# User Guide

For all EduManage users.

## Getting Started
- Login at /login with email + password
- Forgot password? /forgot-password
- New school? /register

## Navigation
Bottom nav on each dashboard. Role-specific tabs.

## Notifications
Bell icon shows unread count. Tap to view. /notifications for full list.

## For Teachers
- Assignments: create, view submissions, grade
- Attendance: mark daily
- Grades: record exam results
- AI Assistant: lesson plans, grading help, quiz generator

## For Students
- Assignments: view + submit
- Grades: view exam results
- Quizzes: take auto-marked quizzes
- Timetable: view weekly schedule
- Fees: view outstanding balance
- AI Tutor: get help with homework

## For Parents
- Switch between children
- View grades, attendance, fees
- Message teachers
- View announcements

## For Bursars
- Invoices: create + track
- Payments: record (M-Pesa, cash, cheque, etc.)
- Receipts: auto-generated
- Reports: financial summaries

## For Librarians
- Books: catalog management
- Borrow/Return: track with auto-fines
- Inventory: stock levels

## For Nurses
- Medical records: student health info
- Clinic visits: log visits
- Medications: administer + track

## For ICT
- Users: manage school members
- Logs: view audit trail
- Domains: manage custom domains

## For Boarding Masters
- Dormitories: manage rooms
- Beds: assign students
- Attendance: nightly check
- Inspections: record scores
